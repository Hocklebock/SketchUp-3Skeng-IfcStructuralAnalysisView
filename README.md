# SketchUp-3Skeng Ifc options-IfcStructuralAnalysisView
## 3Skeng Steelwork analytical model export
The 3Skeng Ifc options allow the users to export the corresponding analytical model to the designed structural model.
An IfcStructuralAnalysisView is derived from 3Skeng Steelwork trusses or frames.
